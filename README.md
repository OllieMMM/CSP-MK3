## Welcome to the Color Shape Program CSP

This has been a long standing project of mine that I want to final implament properly.
I have been working on it for a long time and now with this version MK3, I decided to collate the whole thing, publish it on GitHub and create a functional UI that hopefully someothers can use it and also get inspired by it!

The idea came a long time ago where I wanted to create a cool laser cut puzzle out of differnt pieces of wood, like a jigsaw.
Th eproblem was that I would tedioulsy draw these images by hand and try to attribute the best colour palletes with the limited reasources I had; so as a programmer and negineer what better way to speed up the process then work for hours and hours to create a custom algorithm and codebase to do it for me.

## The Project
Here is the idea:
- Given a finite and unique set of colors, segment and recreate said image. 
- Trace and outline all of these segments (cant be too small or compliacted) into smooth svg's
- Laser cut those svgs and recreate the wooden jigsaw.
